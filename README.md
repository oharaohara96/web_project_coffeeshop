Projeto Coffee Shop - Triple Peaks

Este é um projeto de uma Landing Page responsiva para uma cafeteria temática chamada Triple Peaks. O site foi desenvolvido para fornecer informações sobre receitas de café, permitir a reserva de mesas e facilitar o contato através das redes sociais.

Descrição do Projeto

A página foi construída com foco em semântica HTML e organização de estilos utilizando a metodologia BEM (Block Element Modifier). Ela apresenta:

-Cabeçalho: Com navegação intuitiva e horários de funcionamento.
-Seção de Receitas: Integração com vídeos do YouTube para ensinar métodos de preparo (Aeropress e Prensa Francesa).
-Sistema de Reservas: Um formulário funcional com validação de campos.
-Rodapé: Layout personalizado com elementos decorativos (círculo dinâmico) e links sociais.
